## 📖 Краткое описание

Telegram-бот для удобного просмотра расписания занятий колледжа. Показывает актуальное расписание по курсам, поддерживает интерактивные кнопки и автоматически обновляет данные. Больше не нужно заходить на сайт — всё доступно прямо в мессенджере.

![Demo]([СКРИНШОТ])

## ✨ Возможности

- 📅 Просмотр расписания по курсам (1–4 курс)
- 🔘 Удобная навигация через **inline** и **reply** кнопки
- 🔄 Автоматическое обновление данных раз в сутки
- 💾 Умное кеширование — не скачивает файлы повторно без изменений
- 📊 Поддержка формата `.xlsx` (Excel)

---

## 🛠 Технологический стек

- **Python 3.11+**
- **python-telegram-bot** — работа с Telegram API
- **requests** — HTTP-запросы к API колледжа
- **Vercel** — деплой и хостинг
